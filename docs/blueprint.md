# **App Name**: InstaGalleryAI

## Core Features:

- Instagram Integration: Connect to the user's Instagram account to access posts and highlights.
- Theme Input: Allow the user to input custom themes for gallery creation.
- AI Theme Suggestion: Suggest themes based on current trends and the user's posting history. Provide options for how harsh the AI selection should be.
- Gallery Generation: Generate a gallery of images based on the selected theme, filtering out videos and reels.
- AI-Powered Critique: Provide AI-generated critiques of each image and the overall gallery, considering artistic intention, theme relevance, and AI usage, using an AI tool to decide whether an intention has been respected.
- AI Detection: Detection whether an image is Photography, Plastic Art, etc. Detection whether and how AI has been used in its creation/edition, and provide feedback.
- Critique Report: Generate a report summarizing the AI critique, focusing on areas for improvement.

## Style Guidelines:

- Primary color: Muted teal (#63BDBD), evoking creativity without being distracting.
- Background color: Light grey (#F0F0F0), for a clean, neutral backdrop that puts focus on the user's photos.
- Accent color: Soft coral (#F08080) for interactive elements and highlights, such as the critique report button, adding contrast but not distraction.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look suitable for both headlines and body text.
- Simple, minimalist icons to represent different themes and functions within the app.
- Clean and modern layout with a focus on showcasing the user's images prominently.
- Subtle animations for transitions and loading states to enhance the user experience.
